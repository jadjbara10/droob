## 1. نظرة عامة

لوحة تحكم دروب هي واجهة إدارة بيانات منصة تخطيط الرحلات. تمكنك من:

- 🏷️ إدارة المحطات والخطوط والمسارات (إضافة، تعديل، حذف)
- 📋 مراجعة واعتماد تصحيحات المستخدمين للبيانات
- 📊 عرض إحصائيات الاستخدام
- 📥 استيراد البيانات (GTFS، CSV، Excel)
- 📢 إدارة الإعلانات (لتحقيق الدخل)
- ⚠️ إرسال تنبيهات للمستخدمين

> ⚠️ **مهم:** دروب هو **خدمة مجتمعية لتخطيط الرحلات** وليس نظام إدارة أسطول. لا نملك صلاحيات على المركبات أو السائقين. نحن نقدم للمستخدم خريطة ذكية لتخطيط رحلته من النقطة أ إلى النقطة ب باستخدام خطوط النقل العام الثابتة.

### الأدوار والصلاحيات:

| الدور | الصلاحيات |
|-------|-----------|
| **مدير عام** 👑 | كل شيء — نظام كامل |
| **محرر** ✏️ | إدارة المحطات والخطوط والبيانات |

### الوصول للوحة التحكم:

```
URL: https://droob-dashboard.vercel.app
```

## 2. تسجيل الدخول

```
١. افتح رابط لوحة التحكم
٢. أدخل البريد الإلكتروني وكلمة المرور
٣. اضغط "دخول"
٤. إذا نسيت كلمة المرور، اضغط "نسيت كلمة المرور"
```

### شاشة تسجيل الدخول:
- خلفية داكنة مع شعار دروب
- حقل البريد الإلكتروني
- حقل كلمة المرور
- زر "دخول"
- رابط "نسيت كلمة المرور"

## 4. إدارة المحطات

**المسار:** القائمة الجانبية → المحطات

### عرض المحطات:
- جدول بجميع المحطات (134 محطة حالياً)
- فلترة بالمحافظة
- بحث بالاسم أو الرمز
- أعمدة: الاسم، الرمز، المحافظة، التسهيلات

### إضافة محطة جديدة:

```
١. اضغط "إضافة محطة"
٢. املأ الحقول:
   - الرمز (مثال: ST-AM-11)
   - المحافظة (اختر من القائمة)
   - الاسم بالعربية (مثال: دوار المدينة الرياضية)
   - الاسم بالإنجليزية (مثال: Sports City Circle)
   - التسهيلات: ☑ ملجأ  ☑ إضاءة  ☑ كراسي  ☑ مكيف  ☑ تذاكر
٣. اضغط "حفظ"
```

### تعديل محطة:
- اضغط على أي محطة في الجدول
- ستفتح نافذة التعديل
- عدّل المعلومات المطلوبة
- اضغط "حفظ التغييرات"

### استيراد محطات من ملف:
- CSV أو Excel
- اضغط "استيراد CSV"
- اختر الملف من جهازك
- راجع المعاينة
- اضغط "تأكيد الاستيراد"

## 5. إدارة الخطوط

**المسار:** القائمة الجانبية → الخطوط

### عرض الخطوط:
- جدول بجميع الخطوط (37 خطاً حالياً)
- فلترة بالنوع (باص مدينة، باص سريع، سرفيس، بين المدن)
- فلترة بالحالة (نشط، غير نشط)
- بحث بالاسم أو الرمز

### إضافة خط جديد (قيد التطوير):

```
١. اضغط "إضافة خط"
٢. املأ:
   - نوع الخط (city_bus / brt / serveece / intercity)
   - الاسم والرمز
   - الوكالة (GAM، BRT، JETT، إلخ)
   - محطة الانطلاق ومحطة الوصول
   - السعر الأساسي (دينار)
   - زمن التردد (peak/off-peak بالدقائق)
   - أول وآخر رحلة
٣. اضغط "حفظ"
```

### تعديل خط:
- اضغط على أي صف في الجدول
- عدّل المعلومات مباشرة في الجدول
- التغييرات تحفظ تلقائياً

### إدارة محطات الخط:
- اضغط على خط ← "إدارة المحطات"
- اسحب وأفلت لترتيب المحطات (قيد التطوير)
- أضف أو احذف محطات من الخط

### استيراد GTFS:
- اضغط "استيراد GTFS"
- اختر ملف GTFS.zip
- سيقوم النظام بتحليل الملف وإضافة الخطوط والمحطات والجداول

## 6. إدارة الإعلانات 🆕

**المسار:** القائمة الجانبية → الإعلانات

دروب منصة مجانية للمستخدمين. لتحقيق الاستدامة المالية، نعرض إعلانات في التطبيق:

### أماكن عرض الإعلانات:

| المكان | النوع | الحجم | الظهور |
|--------|-------|-------|--------|
| أسفل الشاشة الرئيسية | Banner | 320×50 | دائم |
| بين نتائج تخطيط الرحلات | Native | بطاقة | كل 3 نتائج |
| أسفل لوحة المغادرات | Banner | 320×50 | كل 10 صفوف |

### إدارة الإعلانات:
- ✅ تفعيل/تعطيل الإعلانات
- 📊 إحصائيات: مرات ظهور، نقرات، إيرادات
- 🎯 استهداف: حسب المحافظة أو المدينة
- 📅 جدولة: أوقات الذروة وغير الذروة

## 7. إدارة التنبيهات

**المسار:** القائمة الجانبية → التنبيهات

### أنواع التنبيهات:

| النوع | اللون | مثال |
|-------|-------|------|
| ⚠️ **تحذير** | 🟡 أصفر | تأخير 10 دقائق على خط ٧ |
| 🚨 **حرج** | 🔴 أحمر | إغلاق محطة الجاردنز |
| ℹ️ **معلومة** | 🔵 أزرق | إضافة خط جديد BRT2 |
| 🔧 **صيانة** | ⚫ رمادي | أعمال صيانة على خط S03 |

### إنشاء تنبيه:

```
١. اضغط "تنبيه جديد"
٢. املأ:
   - العنوان بالعربية (إجباري)
   - الرسالة بالعربية (إجباري)
   - العنوان والرسالة بالإنجليزية (اختياري)
   - مستوى الخطورة (تحذير / حرج / معلومة / صيانة)
   - الخطوط المتأثرة (اختر من القائمة)
   - تاريخ البداية والنهاية
٣. اضغط "إرسال" ← سيظهر التنبيه في تطبيق الهاتف
```

### بث طارئ:
```
١. اضغط "بث طارئ" 🚨
٢. اكتب رسالة الطوارئ
٣. سيظهر عداد تنازلي 3 ثوانٍ
٤. يتم إرسال التنبيه فوراً لجميع المستخدمين
```

## 9. مراجعة مساهمات المجتمع

هذا أهم قسم! هنا تراجع تعديلات المستخدمين على البيانات:

### أنواع المساهمات:

| النوع | الوصف | الإجراء |
|-------|-------|---------|
| 📝 **تصحيح بيانات** | مستخدم يقترح تعديل اسم/سعر/مسار | ✅ اعتماد / ❌ رفض |
| ⏰ **تأخير** | بلاغ عن تأخير | 📋 تسجيل للمراجعة |
| 👥 **ازدحام** | بلاغ عن ازدحام | 📋 تسجيل للمراجعة |

### سير عمل مراجعة التصحيح:

```
١. مستخدم يرسل: "اسم المحطة الصحيح هو دوار الواحة وليس دوار الوaha"
٢. يظهر في لوحة التحكم ← قسم "التنبيهات" أو "المساهمات"
٣. المحرر يراجع الاقتراح
٤. إذا كان صحيحاً ← ✅ اعتماد ← تحديث البيانات
٥. إذا كان خاطئاً ← ❌ رفض ← إشعار للمستخدم بسبب الرفض
```

## 10. التقارير والإحصائيات

**المسار:** القائمة الجانبية → التقارير

### تبويب "نظرة عامة":
- 📊 عدد الرحلات اليوم
- 👥 عدد المستخدمين النشطين
- ⭐ المحطات الأكثر زيارة
- 🚌 الخطوط الأكثر استخداماً

### تبويب "الرحلات":
- 📈 رسم بياني للرحلات بالساعة
- 📊 مقارنة أسبوعية
- 📋 جدول الرحلات المكتملة والملغية

### تبويب "المالية" (لاحقاً):
- 💰 إيرادات التذاكر
- 💰 إيرادات الإعلانات
- 💰 إيرادات الاشتراكات

### تبويب "الأداء":
- ⏱️ نسبة الالتزام بالمواعيد
- ⭐ تقييمات المستخدمين
- 📝 عدد مساهمات المجتمع

### تصدير التقارير:
- 📄 **PDF** — لطباعة التقرير
- 📊 **Excel** — لتحليل البيانات
